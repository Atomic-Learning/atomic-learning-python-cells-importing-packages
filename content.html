Modules from the Python Standard Library

The way interactive Python cells are configured in the Atomic Learning Platform, modules from the Python Standard Library are available for immediate use without any additional setup. import math print(math.sqrt(16))

Packages from Outside the Standard Library

Packages that are not part of the Python Standard Library need to be specified in the metadata of the content item to be available for use in Python cells. This is done by listing the package names under the "python_packages" key in the metadata file. For instance, in the metadata of this content item, the packages numpy and pandas are listed. The end of the metadata file looks like this:
